Teacher Report Assistant

This tool (TRA) is to assist teachers when writing comments about their students by using template sentences that when selected are filled in with the student name and correct gender pronouns. The input files are:

  • names.tra
  • sentenceTemplates.tra
From the menu bar select File → Open directory… to specify a directory that contains these two files.

Input File Formats

names.tra

This file contains the name and gender of each student, one student per line. The first character of each line is M or F to indicate male or female. The students name follows this initial character. Here is an example:
   M Donald Duck
   F Minnie Mouse
   M Bugs Bunnie
   F Flipper

sentenceTemplates.tra

This file contains the sentence templates. Any occurrences of _NAME in the template are replaced with the student's name. Similarly, any occurrence of _HE, _HIM, and _HIS are replaced with the gender appropriate pronoun.

The sentence templates are in categories. Example categories might be:

  • Current Performance
  • Class Work
  • Homework
  • Conclusion
A new category is indicated by a line that begins with ". Here is an example:
   "Current Performance
   _NAME is do extremely well.
   _Name is not performing to _HIS potential.
  
   "Class Work
   _HIS classwork is always excellent.
   _NAME is not always focused so _HIS classwork suffers.
  
   "Homework
   It is obvious that _NAME takes great pride in _HIS homework.
   Too often _NAME has not completed _HIS homework.
  
   "Conclusion
   It is a pleasure having _NAME in class.
   _NAME should seek extra help to improve _HIS current grade.
 Please see the directory traSampleFiles for another example of these files.

Creating Reports

There are left and right arrows to advance to the next student or return to a prior student. There is a tab for each category of comments. On the tabbed pane select the desired comments to be included in the report. The student reports are editable on the window.

Writing the Reports

From the menu bar select File → Save as… to specify the file to be written with all student reports.
